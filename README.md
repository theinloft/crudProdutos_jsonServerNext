# 📦 Cadastro de Produtos - CRUD

Aplicação CRUD para gerenciamento de produtos desenvolvida com **Next.js**.

## 🚀 Tecnologias Utilizadas

* Next.js
* React
* Node.js
* npm

## 📋 Pré-requisitos

Antes de executar o projeto, certifique-se de ter instalado:

* Node.js (versão 18 ou superior recomendada)
* npm

## ⚙️ Instalação

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd cadastroproduto-crud
```

3. Instale as dependências:

```bash
npm install
```

## ▶️ Executando a Aplicação

Após a instalação das dependências, inicie o ambiente de desenvolvimento:

```bash
npm run dev:full
```

## 🌐 Acessando a Aplicação

Com a aplicação em execução, abra o navegador e acesse:

```text
http://localhost:3000/produtos
```

## 📌 Comandos Úteis

| Comando            | Descrição                                     |
| ------------------ | --------------------------------------------- |
| `npm install`      | Instala todas as dependências do projeto      |
| `npm run dev:full` | Inicia a aplicação em modo de desenvolvimento |
